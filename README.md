# Ayman Ta'lim — React sayt

## Ishga tushirish
```
npm install
npm run dev
```
Brauzerda: http://localhost:5173

## Build (production)
```
npm run build
```
Natija `dist/` papkasida hosil bo'ladi, uni istalgan hostingga joylashtirish mumkin.

## Tuzilma
- `src/Global.css` — barcha stillar shu yerda
- `src/pages/` — Home, Kurslar, BizHaqimizda, Aloqa sahifalari
- `src/components/` — Header, Footer, Reveal (scroll animatsiyasi)
- `src/App.jsx` — react-router-dom orqali sahifalar aro navigatsiya
